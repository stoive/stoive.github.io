The W3C says HTML5 is:

  • new semantic elements
  • offline storage and application caching
  • device access
  • push data to browser
  • multimedia
  • flashy (ahem) graphics
  • performance (multitasking!) and integration (better XHRs!)
  • css3

Of the above, five don't even involve HTML markup at all!
